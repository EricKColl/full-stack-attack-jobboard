# Checklist del mockup - Producto 1 (JobConnect)

## Requisitos generales del mockup
- [ ] Diseño limpio y consistente en las 4 pantallas
- [ ] Barra de navegación igual en todas las pantallas
- [ ] Uso coherente de colores (ofertas y demandas diferenciadas)
- [ ] Distribución responsive (desktop y móvil, aunque sea esquemático)
- [ ] Botones con estados claros (guardar, eliminar, login)
- [ ] Formularios claros con labels
- [ ] Tablas/listados legibles
- [ ] Jerarquía visual correcta (títulos, subtítulos, bloques)

## Pantalla 1 - Dashboard
- [ ] Navbar
- [ ] Hero principal con nombre de la app
- [ ] Tarjetas resumen (nº ofertas, demandas, usuarios)
- [ ] Grid de tarjetas de publicaciones
- [ ] Colores distintos para ofertas y demandas
- [ ] Footer

## Pantalla 2 - Login
- [ ] Navbar
- [ ] Formulario centrado
- [ ] Campos email y contraseña
- [ ] Botón iniciar sesión
- [ ] Zona para mensajes de error/éxito

## Pantalla 3 - Gestión de ofertas y demandas
- [ ] Navbar
- [ ] Formulario completo de alta
- [ ] Selector tipo oferta/demanda
- [ ] Botón guardar
- [ ] Tabla/listado de publicaciones
- [ ] Botón eliminar por fila

## Pantalla 4 - Gestión de usuarios
- [ ] Navbar
- [ ] Formulario alta usuario
- [ ] Campos nombre, apellidos, email, contraseña, rol
- [ ] Botón crear usuario
- [ ] Tabla de usuarios
- [ ] Botón eliminar por fila
