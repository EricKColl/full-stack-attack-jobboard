# FULL STACK ATTACK - JobBoard (Producto 1)

## Descripción
Proyecto del grupo **FULL STACK ATTACK** para el desarrollo de una plataforma de ofertas y demandas de empleo.

En este **Producto 1** se desarrolla el frontend inicial con:
- HTML5
- CSS3
- Bootstrap
- JavaScript (básico y modular)

## Objetivo del producto
Crear un prototipo funcional de la aplicación con las siguientes interfaces:
1. Dashboard principal
2. Login de usuario
3. Gestión de ofertas y demandas
4. Gestión de usuarios

## Tecnologías
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES Modules)
- Git / GitHub
- Trello (planificación)

## Estructura del proyecto
- `index.html` → Dashboard
- `login.html` → Login
- `ofertas-demandas.html` → Gestión de ofertas y demandas
- `usuarios.html` → Gestión de usuarios
- `assets/css/styles.css` → Estilos personalizados
- `assets/js/datos.js` → Datos comunes (arrays y objetos JS)
- `assets/js/*.js` → Lógica por página
- `docs/` → Documentación de IA, bibliografía, arquitectura, mockups
- `evidencias/` → Mapa conceptual y evidencias
